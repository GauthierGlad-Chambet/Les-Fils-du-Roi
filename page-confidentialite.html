Politique de confidentialité


Votre vie privée est une priorité pour nous. Cette politique de confidentialité explique comment 'Les Fils du Roi' collecte, utilise et protège vos informations personnelles lorsque vous utilisez notre site web https://www.lesfilsduroi.fr.

Introduction

La présente Politique de confidentialité a pour objectif de vous informer de manière transparente sur les données personnelles que nous collectons, utilisons, et protégeons lorsque vous utilisez le site 'Les Fils du Roi', accessible à l'adresse suivante : https://www.lesfilsduroi.fr. Nous nous engageons à protéger vos informations personnelles et à respecter votre vie privée.
Cette politique s'applique à tous les utilisateurs du site, quel que soit leur statut ou leurs interactions avec le site (navigation, inscription, achat, etc.).

Identité du responsable du traitement des données

Le responsable du traitement des données personnelles collectées sur ce site est :
Nom de l'éditeur : Gauthier Glad-Chambet

Types de données collectées

Nous collectons différentes catégories de données personnelles lorsque vous utilisez notre site :

  • Données d'identification : Nom, prénom, adresse e-mail, adresse postale, numéro de téléphone, etc.
  • Données de connexion : Adresse IP, type de navigateur, système d'exploitation, date et heure de la visite, pages consultées, etc.
  • Données de navigation : Historique de navigation sur le site, préférences de l'utilisateur, cookies, etc.
  • Droit à la limitation du traitement : Informations de carte bancaire (ces données sont traitées par des prestataires de paiement sécurisés et ne sont pas stockées directement sur notre site).


Finalités du traitement

Nous utilisons vos données personnelles pour :

  • Répondre à vos demandes via le forumulaire de contact.
  • Améliorer votre expérience utilisateur.
  • Analyser le trafic sur notre site web pour optimiser nos services.


Base légale du traitement

Le traitement de vos données repose sur les bases légales suivantes :

  • Votre consentement lorsque vous remplissez un formulaire ou acceptez les cookies.
  • L'intérêt légitime pour l'analyse et l'amélioration de nos services.


Partage des données

Vos données personnelles ne seront jamais vendues. Toutefois, elles peuvent être partagées avec :

  • Nos prestataires techniques pour assurer le bon fonctionnement du site (ex. hébergement).
  • Les autorités publiques en cas d'obligation légale.


Durée de conservation des données

Nous conservons vos données uniquement le temps nécessaire aux finalités pour lesquelles elles ont été collectées :

  • Données du formulaire de contact : 1 an après la dernière interaction.
  • Cookies : Voir notre Politique de Cookies pour les durées spécifiques.


Vos droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : Demander une copie de vos données personnelles.
  • Droit de rectification : Corriger vos informations si elles sont incorrectes.
  • Droit à l'effacement : Supprimer vos données, sauf obligation légale contraire.
  • Droit à la limitation du traitement : Restreindre l'utilisation de vos données.
  • Droit à la portabilité : Recevoir vos données dans un format structuré.
  • Droit d'opposition : Vous opposer à l'utilisation de vos données.

Pour exercer vos droits, contactez-nous à lesfilsduroi@outlook.fr. Vous pouvez également introduire une réclamation auprès de la CNIL (www.cnil.fr).


Sécurité des données

Nous mettons en place des mesures techniques et organisationnelles pour protéger vos données contre toute perte, vol ou accès non autorisé.


Cookies

Consultez notre Politique sur les Cookies pour plus d'informations sur l'utilisation des cookies sur notre site.
